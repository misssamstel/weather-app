# WeatherApp
Practice weather application using React, Webpack and Babel

## Install
```
yarn install
```

## Develop
### Watch with babel
```
yarn run dev-server
```
### Build
```
yarn run build
```

## History
* v0 - First Steps
* v1 - State Management

## Weather API
### AccuWeather
https://developer.accuweather.com/

### Limitations
Only certain APIs available with free package. (Forecast 5-day only)
Limited to 50 API calls a day.
